Important HTML tags

This page is designed as a quick reference for some of the common tags covered in Learn Enough HTML to Be Dangerous. In the process of making it, we'll learn how to make HTML tables via table and related tags.

Tahe tables below don't include all html tags, but they do list many of the most important ones.

Block Elements




Tag Name Purpose
h1–h6 headings including a heading (levels 1–6)
p paragraph include a paragraph of text
table table include a table
tr table row include a row of data
th table header make a table header
td table data include a table data cell
header header opening block preceding code
div div block element for styling
ol ordered list a numbered list
ul unordered list a list with bullet points
blockquote blockquote A block indicating a quote
.
